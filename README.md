# ProcMon

A simple process monitor that with a neat GUI offering three different types of views

## Views

### Top View
	Default view that lists the top 20 CPU intensive processes (press 'r' for Memory intensive)

### Anomaly View
	View that lists processes whose maximum CPU usage is more than twice it's average

### Miscellaneous View
	Lists zombie processes, long running and Frequently run processess (Still WIP)
